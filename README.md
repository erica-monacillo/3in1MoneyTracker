# 3-in-1 Money Tracker
Demo: https://3in1-money-tracker.vercel.app

A personal finance app concept that combines three essential tools in one clean and user-friendly experience: Cash Flow Tracker, Dues Ko Po! Bayarin Tracker, and Mission Impossible Ipon Challenge.

## Features
- Track monthly cash flow with income, expenses, and running balance
- Monitor bills and payment status by month
- Join savings challenges with visual progress tracking
- View saved amount, remaining goal, and suggested daily/weekly savings targets

## Goal
This project was designed to make budgeting, bill tracking, and saving money easier, simpler, and more motivating for everyday users.

## Tools
- Figma for UI/UX design
- Supabase-ready structure for future backend integration

  ## Running the code
  Run `npm i` to install the dependencies.
  Run `npm run dev` to start the development server.
